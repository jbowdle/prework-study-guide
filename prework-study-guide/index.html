HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Metadata elements go in the head and features data about data.
  • Charset element sets character encoding.
  • http-equiv elements set an HTTP header for the content.
  • Viewport elements set the default view for the webpage.
  • Semantic elements are elements where the name describes its content and role.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Webpages can be styled with inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline style occurs directly in the HTML file, offering a quick and permanent change.
  • Internal style sheets are rules written in the head section and only apply to that page.
  • An external style sheet is a separate CSS file that can be applied to a whole website.
  • A CSS rule contains a selector and declaration block.
  • A declaration block is made of declarations which are made up of a property and a value.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • A branch can be created to allow a space to work without affecting the default branch.
  • git add -A: 'add' adds modifications in the current working branch to the staging area. -A indicates that all changes will be added.
  • git commit -m '': This will create a commit of the currently staged changes.
  • git push origin (branch): This will push the local changes to the remote GitHub branch.
  • git pull origin main: This let's us check whether our local version of the project is up to date with the remote GitHub version.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A method is a set of instructions that can be executed by a computer.
  • The console.log() method will output its argument to the console.
  • An if statement is a conditional statement that introduces a decision making element to code.
  • Truthy and falsy values are values that are considered true or false in a Boolean context.
  • Falsy values include 0, -0, On, "", null, undefined, NaN, and false. All other values are truthy.
  • An array is a group of related data that is all stored together in one variable.
  • Array indexes start at 0.
  • Using a for loop is an efficient way to perform a task on all items in an array.
  • A function is a set of instructions that tells a computer how to store a specific task.
  • A function has to be defined then called for it to execute. This means it is reusable because it can be called multiple times.
